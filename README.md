Contains the following components Button, CheckBox, Image, Label, ListItem, TextBox
Controls can be grouped together. You can link to an object.

=First launch of the project=
Rebild the files in Content.mgcb
And install the extension <i> MonoGame Framework C# project templates</i>

<b>*</b> Now settings can be saved to an XML file
<b>*</b> Any control will be able to use both animated and static textures.
<b>*</b> There are events to configure actions


AnchorLocation - Automatic alignment in the parent group by parameters (similar WPF):
1/ Horizontal Alignment: Center, Left, Right, Full
2/ Vertical Alignment: Top, Bottom, Center, Full 
3/ Height and Width: default sizes
4/ Position - indents, first aligns according to the argument, then indents are taken 
5/ ListItemUI - A wide items of elements
